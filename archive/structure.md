# Page Hierarchy & Routing

## Pages
- `/` - Home: Hero, parallax, value cards.
- `/about` - Company history, creator photos, and portfolio gallery.
- `/articles` - News grid fetched from `articles.json`.
- `/admin` - Password-protected dashboard for content updates.

## Components
- `Layout`: Persistent Navbar and Footer.
- `ThemeToggle`: Icon button for switching modes.
- `ArticleCard`: Individual card with animations.
- `AdminForm`: Visual editor for articles.
